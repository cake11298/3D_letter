# Gustave's Interactive 3D Portfolio

一個真正有質感的 3D 互動履歷網站。

## 特色

### 🎮 真實的 3D 體驗
- **厚重的文件感**：30 頁疊加的紙張，每頁都有物理厚度
- **立體按鈕**：可以按下、彈起的 3D 紅色按鈕
- **戲劇性打光**：聚光燈、邊緣光、重點光營造專業感
- **材質細節**：金屬、紙張、發光效果

### 🎨 互動功能
- 🖱️ **拖動旋轉**：滑鼠拖動可 360° 查看文件
- 🔴 **點擊翻頁**：點擊紅色按鈕翻到下一頁
- ⌨️ **鍵盤控制**：方向鍵或空白鍵翻頁
- 📄 **頁面動畫**：流暢的翻頁效果

### 💡 視覺效果
- 漂浮動畫
- 按鈕呼吸燈效果
- 陰影與反射
- 後處理渲染

## 技術棧

- **Three.js** - 3D 圖形渲染
- **GSAP** - 動畫庫
- **Webpack** - 模組打包
- **ES6+** - 現代 JavaScript

## 開發

```bash
# 安裝依賴
npm install

# 開發模式（熱重載）
npm run dev

# 構建生產版本
npm run build
```

這不是一個「網頁式」的履歷，而是一個可以玩的 3D 模型。

- 真實的物理感（厚度、重量、光影）

就像 Gustave 本人：不走尋常路，但做得很專業。
